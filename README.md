# JSON Viewer — 내부 전용 로컬 JSON 뷰어

> **오프라인 전용 도구입니다.** 외부 서버, 인터넷, 또는 서드파티 서비스와 일절 통신하지 않습니다.

---

## 프로젝트 개요

`json-viewer.html`은 브라우저에서 완전히 동작하는 단일 HTML 파일입니다.  
JSON 파일을 불러오거나 텍스트를 붙여넣으면 읽기 쉬운 표(테이블) 형태로 시각화해 줍니다.

이 도구는 내부 업무용으로 설계되었으며, 다음과 같은 환경에 적합합니다:

- 인터넷이 차단된 폐쇄망 환경
- 외부 라이브러리나 CDN 사용이 금지된 보안 환경
- 서버 설치 없이 누구나 즉시 사용해야 하는 경우

---

## 주요 기능

| 기능 | 설명 |
|---|---|
| JSON 파일 로드 | `.json` 파일을 브라우저에서 직접 읽어 들입니다 |
| 텍스트 붙여넣기 | JSON 텍스트를 직접 입력 창에 붙여넣어 분석합니다 |
| 테이블 렌더링 | 객체 배열을 자동으로 표로 변환합니다 |
| 중첩 구조 펼치기 | 중첩된 객체를 점(`.`) 표기법으로 펼쳐서 열로 표시합니다 |
| 컬럼 자동 추론 | 모든 행을 스캔하여 누락된 열도 자동으로 감지합니다 |
| 유효성 검사 | 잘못된 JSON 입력 시 오류 위치를 포함한 메시지를 표시합니다 |
| 행/열 수 표시 | 파싱된 데이터의 행 수, 열 수, 타입을 상단에 표시합니다 |
| 고정 헤더 | 테이블 스크롤 시 헤더가 상단에 고정됩니다 |
| 초기화 버튼 | 입력과 결과를 한 번에 초기화합니다 |

---

## 사용 방법

### 방법 1 — JSON 파일 불러오기

1. **Load JSON file** 버튼을 클릭합니다.
2. 컴퓨터에서 `.json` 파일을 선택합니다.
3. 파일이 자동으로 입력 창에 로드됩니다.
4. **Parse & View** 버튼을 클릭합니다.

### 방법 2 — JSON 텍스트 붙여넣기

1. 분석할 JSON 텍스트를 복사합니다.
2. 입력 창(텍스트 영역)에 붙여넣습니다.
3. **Parse & View** 버튼을 클릭하거나 `Ctrl+Enter`를 누릅니다.

### 결과 해석

| JSON 형식 | 표시 방식 |
|---|---|
| 객체 배열 `[{…}, {…}]` | 다중 행 테이블 |
| 단일 객체 `{…}` | 1행짜리 테이블 |
| 값 배열 `["a", 1, true]` | 단일 열 테이블 |
| 기타 값 (문자열, 숫자 등) | 텍스트로 표시 |

### 셀 색상 안내

| 색상 | 의미 |
|---|---|
| 기울임체 회색 | `null` 값 |
| 보라색 굵은 글씨 | `true` / `false` (불리언) |
| 파란색 | 숫자 |
| 주황색 기울임체 | 배열 (미리보기; 전체 내용은 마우스를 올려 확인) |
| 연한 회색 줄표(—) | 해당 행에 존재하지 않는 컬럼 |

## 실행 방법

별도의 설치, 서버, 또는 인터넷 연결이 필요하지 않습니다.

1. `json-viewer.html` 파일을 아무 위치에 저장합니다.
2. 파일을 더블클릭하거나, 브라우저에 드래그하여 엽니다.
3. 주소 표시줄에 `file:///경로/json-viewer.html` 형식으로 표시됩니다.
4. 즉시 사용 가능합니다.

**지원 브라우저:** Chrome, Edge, Firefox, Safari (최신 버전 권장)

## 보안 설명 — 외부 통신 없음

이 도구는 보안을 최우선으로 설계되었습니다.

### 외부 통신이 없는 이유

| 항목 | 상태 | 설명 |
|---|---|---|
| 외부 스크립트 로드 | 없음 | CDN, npm, 원격 `<script src>` 일절 없음 |
| 외부 CSS/폰트 | 없음 | 구글 폰트 등 외부 리소스 없음. 시스템 폰트만 사용 |
| 네트워크 요청 | 없음 | `fetch`, `XMLHttpRequest`, `WebSocket` 등 사용 안 함 |
| 분석/추적 도구 | 없음 | Google Analytics 등 일절 없음 |
| 원격 폼 전송 | 없음 | `<form action>` 없음 |
| 콘텐츠 보안 정책(CSP) | 적용됨 | `default-src 'none'` 으로 모든 외부 요청을 브라우저 수준에서 차단 |

### 데이터가 안전한 이유

- 파일은 브라우저의 **FileReader API**를 통해 로컬 디스크에서 직접 읽힙니다.  
  파일 내용이 서버에 업로드되거나 네트워크로 전송되지 않습니다.
- 모든 JSON 값은 `textContent`로만 DOM에 삽입됩니다.  
  HTML로 해석되지 않으므로, 악의적인 문자열(`<script>…</script>`)도 화면에 텍스트로만 표시됩니다.
- `eval()` 함수는 사용되지 않습니다. JSON 파싱은 안전한 `JSON.parse()`만 사용합니다.

### 파일 무결성 확인 방법 (선택 사항)

배포 시 파일의 SHA-256 해시를 함께 공유하면 수신자가 변조 여부를 확인할 수 있습니다.

```bash
# Windows
certutil -hashfile json-viewer.html SHA256

# macOS / Linux
sha256sum json-viewer.html
```

## 파일 구조 설명

이 도구는 **단 하나의 파일**로 구성됩니다.

```
json-viewer.html
├── <head>
│   ├── Content-Security-Policy 메타 태그  ← 브라우저 수준 외부 통신 차단
│   └── <style>                            ← 모든 CSS (인라인)
├── <body>
│   ├── <header>                           ← 제목 및 오프라인 배지
│   ├── <main>
│   │   ├── .input-panel                   ← 파일 로더 + 텍스트 입력 영역
│   │   ├── #status                        ← 오류/경고/성공 메시지
│   │   └── #output-panel                  ← 테이블 또는 텍스트 출력
│   └── <footer>
└── <script>                               ← 모든 JavaScript (인라인)
    ├── 상수 및 DOM 참조
    ├── 파일 로드 처리 (FileReader)
    ├── parseAndRender() — 파싱 진입점
    ├── renderArray() — 객체 배열 → 테이블
    ├── renderScalarArray() — 값 배열 → 단일 열 테이블
    ├── renderScalar() — 기본값 → 텍스트
    ├── buildTable() — DOM 테이블 생성
    ├── flattenObject() — 중첩 객체 평탄화
    ├── setCellContent() — 셀 값 안전 삽입
    └── 유틸리티 함수들
```

외부 파일(이미지, 폰트, 스크립트, 스타일시트)이 전혀 없어 **하나의 파일만 배포하면 됩니다.**

## 주의사항 및 제한사항

### 입력 크기 제한

| 제한 항목 | 기본값 | 이유 |
|---|---|---|
| 최대 입력 크기 | 50 MB | 브라우저 탭 멈춤 방지 |
| 최대 표시 행 수 | 10,000행 | DOM 노드 과다 생성 방지 |
| 최대 표시 열 수 | 200열 | 화면 렌더링 성능 유지 |
| 최대 중첩 깊이 | 20단계 | 호출 스택 오버플로우 방지 |

한도를 초과하면 데이터가 자동으로 잘리고, 화면 상단에 경고 메시지가 표시됩니다.  
한도 값은 파일 내 소스 코드 상단의 상수(`MAX_INPUT_BYTES`, `MAX_ROWS`, `MAX_COLS`, `MAX_DEPTH`)를 직접 수정하여 조정할 수 있습니다.

### 기타 주의사항

- **JSON 형식만 지원합니다.** CSV, XML, YAML 등 다른 형식은 지원하지 않습니다.
- **배열 값은 셀 내에서 미리보기로 표시됩니다.** 마우스를 해당 셀 위에 올리면 전체 내용을 확인할 수 있습니다.
- **브라우저 확장 프로그램**은 이 도구의 CSP 정책을 우회할 수 있습니다. 보안이 중요한 환경에서는 확장 프로그램이 없는 전용 브라우저 프로파일을 사용하세요.
- `file://` 프로토콜로 열 경우 일부 오래된 브라우저에서 FileReader가 제한될 수 있습니다. 최신 Chrome 또는 Edge 사용을 권장합니다.
- 이 파일 자체가 변조되면 악성 코드가 실행될 수 있습니다. **배포된 파일의 해시를 반드시 관리하세요.**

## 향후 개선 가능 사항

아래는 현재 버전에서 구현되지 않은 기능으로, 필요 시 추가 개발이 가능합니다.  
모든 개선은 오프라인 단일 파일 원칙을 유지하는 방향으로 구현되어야 합니다.

| 개선 항목 | 설명 |
|---|---|
| 검색 및 필터 | 특정 값이나 열을 실시간으로 검색하는 기능 |
| 열 정렬 | 열 헤더 클릭으로 오름차순/내림차순 정렬 |
| CSV 내보내기 | 테이블을 CSV 파일로 저장 (Blob API 활용, 외부 통신 없음) |
| 페이지네이션 | 행 수가 많을 때 페이지 단위로 나눠 표시 |
| 열 가시성 제어 | 특정 열을 숨기거나 다시 표시하는 기능 |
| JSON 경로 탐색 | 중첩 구조를 트리 형태로 탐색하는 보조 뷰 |
| 다크 모드 | 시스템 설정에 따른 다크/라이트 테마 전환 |
| 긴 셀 내용 전체 보기 | 셀 클릭 시 전체 값을 모달 또는 사이드 패널로 표시 |
| 대용량 데이터 가상 스크롤 | 수십만 행을 DOM 없이 처리하는 가상화 렌더링 |
